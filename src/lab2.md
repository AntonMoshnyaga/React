1. Atomic Design — це методологія побудови інтерфейсів, де компоненти поділяються на рівні: атоми (базові елементи — кнопки, інпути), молекули (комбінації атомів) тощо.

2. CSS Modules автоматично генерують унікальні імена класів для кожного компонента.
Це запобігає конфліктам стилів, оскільки класи не є глобальними, а локальними для конкретного файлу.

3. props.children — це властивість у React, яка дозволяє передавати вкладений контент всередину компонента. Ми використовували його у компоненті-обгортці (Card).

4.  class є зарезервованим словом у JavaScript, а JSX базується на JavaScript. Тому в React використовується className.
